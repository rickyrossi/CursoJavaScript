O que são variáveis

Formas de declarar variável no JS

var: declara uma variável podendo atribuir um valor.
let: declara uma variável com escopo de bloco.
const: declara uma variável apenas de leitura. Não será possível alterar.

Tipos de Dados

O JavaScript possui os seguintes tipos de dados:

  • Number
  • String
  • Boolean
  • Array
  • Null & Undefined
  • Symbol
  • Object
  • Function
Saber este tipo de informação é importante para evitar erros.

Numbers

São todos os números reais que aprendemos com a matemática.

Operadores: você pode usar + - * e / para executar as operações básicas com números.
